How to use the Pomodoro Timer

  1. Add tasks using the '+' button
  2. Set pomodoro and break lengths in the setitngs (or use the default values)
  3. Select a task to focus on using the magnifying glass icon
  4. Start the timer and be productive!
  5. Take a break when the alarm rings
  6. Repeat steps 3-5 to satisfaction

Features

  • Dark mode theme for late night work sessions
  • Audio notifications at end of pomodoro sessions
  • Customizable pomodoro and break intervals
  • Ability to focus, mark as completed, and delete tasks
  • Focus mode to eliminate page distractions and allow for greater focus on current task

Keyboard Shortcuts

  • 'h' - Help page pop up
  • ';' - Settings pop up
  • 'r' - Reset pop up
  • 'f' - Focus mode
  • 'a' - Add task
  • 'Space' - Start/stop
  • 'Enter' - Confirm/Add
  • 'Esc' - Cancel/Close
